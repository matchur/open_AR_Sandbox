load_frame(frame: ndarray | None = None, from_file: str | None = None)[código fonte]¶

During the sandbox thread, if you want to fix a frame but not interrupt the thread, load the desired numpy array here. This will change the flag self._loaded_frame = False to True in the update function and stop the sensor frame acquisition. # TODO: Make this compatible with DEM of differ shapes (e.g. bennisson DEM) To stop this, pass: frame = None or change the flag self._loaded_frame to False. Args:

frame: numpy.ndarray: must be a matrix of desired resolution from_file: Path to DEM. Can be either .npz or .npy (internally normalized)

Returns:
